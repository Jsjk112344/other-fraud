<!-- Backend SSE event contract (from Plan 01). Frontend must consume these. -->

SSE endpoint: POST /api/investigate
Request body: { "url": "string" }
Response: text/event-stream

Each SSE event has:
- event field: step ID string (e.g., "extract_listing", "verdict", "error")
- data field: JSON string of InvestigationEvent

```typescript
// Types the frontend must define to mirror backend:
interface InvestigationEvent {
  step: string;
  status: 'pending' | 'active' | 'complete' | 'error';
  data?: Record<string, unknown>;
}

interface VerdictResult {
  category: 'LEGITIMATE' | 'SCALPING_VIOLATION' | 'LIKELY_SCAM' | 'COUNTERFEIT_RISK';
  confidence: number; // 0-100
  reasoning: string;
  signals: Array<{ name: string; severity: string; segmentsFilled: number }>;
}

// 6 investigation steps in order:
// extract_listing, investigate_seller, verify_event, check_market, cross_platform, synthesize
```

SSE event sequence for a successful investigation:
1. event: extract_listing, data: {step: "extract_listing", status: "active", data: null}
2. event: extract_listing, data: {step: "extract_listing", status: "complete", data: {...listing data}}
3-12. Same pattern for remaining 5 steps
13. event: verdict, data: {step: "verdict", status: "complete", data: {...verdict with category, confidence, reasoning, signals}}

For invalid URL: event: error, data: {error: "Could not detect listing..."}

    1. Create `frontend/src/api/investigate.ts`:
       ```typescript
       import { fetchEventSource } from '@microsoft/fetch-event-source';
       import type { InvestigationEvent } from '../types/investigation';

       export async function startInvestigation(
         url: string,
         onEvent: (event: InvestigationEvent) => void,
         onError: (error: Error) => void,
         signal: AbortSignal
       ): Promise<void> {
         await fetchEventSource('/api/investigate', {
           method: 'POST',
           headers: { 'Content-Type': 'application/json' },
           body: JSON.stringify({ url }),
           signal,
           onmessage(ev) {
             try {
               const data = JSON.parse(ev.data) as InvestigationEvent;
               onEvent(data);
             } catch (e) {
               onError(new Error('Failed to parse SSE event'));
             }
           },
           onerror(err) {
             onError(err instanceof Error ? err : new Error('SSE connection error'));
             throw err; // stop retrying
           },
         });
       }
       ```

    2. Create `frontend/src/hooks/useInvestigation.ts`:
       - State: `investigationState: InvestigationState` (idle | running | complete | error)
       - State: `steps: StepWithState[]` initialized from INVESTIGATION_STEPS with status='pending'
       - State: `verdict: VerdictResult | null`
       - State: `error: string | null`
       - Ref: `abortControllerRef: React.MutableRefObject<AbortController | null>`

       - `start(url: string)`: Creates AbortController, resets steps to pending, sets state to running, calls startInvestigation. On each event:
         - If event.step === "error": set error state with event.data.error message, set investigationState to error

       - `cancel()`: Calls abortController.abort(), resets all state to idle (steps back to pending, verdict null, error null)
       - Return: { state: investigationState, steps, verdict, error, start, cancel }
